1. Если хотябы один элемен массива равен 0 - TRUE
2. Если нет елемента со значением 0 и все элементы строго одного знака - FALSE
3. Если min(count_positive, count_negatve) > log2(2^31) - TRUE


Решение с полиномиальной сложностью
O(n*w) где w = min(sum(positive), sum(negative))
w не может быть больше чем 2^30 иначе решение задачи возможно за линейное время согласно условию 3.
Оценка для произвольной разрядности элемена массива
w = E[Ai]*n
Пусть r это разрядность
тогда E[Ai] = 2^(r-1), w = n*2^(r-1) и
сложность алгоритма = O(n^2)
